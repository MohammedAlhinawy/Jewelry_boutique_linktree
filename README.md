# Jewelry Boutique Tanzania - Linktree

A beautiful, modern linktree page for Jewelry Boutique Tanzania featuring pink ombre design, floating product icons, and smooth animations.

## 🌟 Features

- **Pink Ombre Design** - Elegant gradient from deep folly to light misty rose
- **Floating Product Icons** - Animated icons representing jewelry, clothing, handbags, perfumes, and shoes
- **Pulsing Logo Animation** - Smooth scaling effect on brand logo
- **Sparkle Effects** - Multiple animated sparkles with varied colors
- **Particle System** - Floating particles for depth and movement
- **Aurora Background** - Dynamic gradient animations
- **Responsive Design** - Optimized for mobile and desktop
- **Social Links** - Instagram, TikTok, Snapchat, WhatsApp, and website
- **Contact Information** - Email, phone, and location details

## 🛠️ Development

### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/MohammedAlhinawy/Jewelry_boutique_linktree.git
   cd Jewelry_boutique_linktree
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 🎯 Customization

### Colors
Edit the CSS variables in `src/App.css`:
```css
:root {
  --folly: #ff0a54ff;
  --french-rose: #ff477eff;
  /* ... other color variables */
}
```

### Links
Update the `socialLinks` array in `src/App.jsx`:
```jsx
const socialLinks = [
  { title: "Your Link", subtitle: "Description", url: "https://example.com", icon: YourIcon },
  // Add more links as needed
];
```

### Product Icons
Modify product icon classes in `src/App.jsx` and corresponding CSS in `src/App.css`:
```jsx
<Gem className="product-icon necklace" />
<ShoppingBag className="product-icon handbag" />
// Add more icons as needed
```
